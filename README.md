# Vortex-MacroPad
A code for my MacroPad. 4 key keyboard with cusomization.

Wiring Note:
| XIAO RP2040 Pin | Connected Device | Signal / Function        |
|-----------------|------------------|--------------------------|
| D0              | Rotary Encoder   | Output A                 |
| D1              | Rotary Encoder   | Output B                 |
| D2              | Rotary Encoder   | Push Button / Switch     |
| D3              | SK6812 MINI      | DIN (Data In)            |
| D4              | OLED Display     | SDA                      |
| D5              | OLED Display     | SCL                      |
| D7              | Switch           | Switch 1                 |
| D8              | Switch           | Switch 2                 |
| D9              | Switch           | Switch 3                 |
| D10             | Switch           | Switch 4                 |
